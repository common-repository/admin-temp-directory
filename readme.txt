== Description ==

A very simple plugin that lists files (with download links) within up to three directories depending on user's capabilities in WordPress admin area.

* All users
* Users with a certain capability
* Admin

Designed for use on single blogs. If you need multisite functionality, contact me.

Developed for private use, but has perspective for more extensive usage. I can't guarantee any support in the future nor further development, but it is to be expected. Kindly inform me about bugs, if you find any, or propose new features: zaantar@zaantar.eu.

== Installation ==

1. Upload `admin-temp-directory.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
